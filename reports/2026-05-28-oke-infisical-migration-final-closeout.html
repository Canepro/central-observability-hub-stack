Deferred Or Non-Consumer Items

These were not forced into Infisical because doing so would weaken ownership clarity or migrate something without a live consumer.

  • signalforge-agent-regcred: not referenced by the live deployment imagePullSecrets.
  • Argo cluster/repo Secrets: Argo-managed control-plane access, deferred.
  • Argo runtime-generated Secrets: bootstrap/session internals, deferred.
  • cert-manager/TLS Secrets: generated by cert-manager, deferred.
  • Helm release Secrets: release metadata, not secret-manager payloads.
  • OCI Vault values were left available as rollback material; deleting old provider values is a separate cleanup gate.
